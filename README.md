# Framework_TET-CVTL__Protezione_Topologica_Eterna_e_Braiding_nel_Grafene_Bilayer
Framework TET-CVTL (Topological Eternal Trefoil – Chiral Vacuum Torque Loop): Protezione topologica eterna e braiding di anyoni Fibonacci/parafermioni in twisted bilayer graphene per la generazione di torque chirale persistente dal vuoto quantistico e applicazioni alla propulsione propellantless interstellare.
